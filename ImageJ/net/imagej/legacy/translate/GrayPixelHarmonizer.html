    public class GrayPixelHarmonizer

    Supports bidirectional synchronization between ImagePluses and gray Datasets. Single channel color ImagePluses are not supported here. But multichannel color ImagePluses are handled and treated as gray data.

      • updateDataset

        public void updateDataset(Dataset ds,
                          ImagePlus imp)
        Assigns the data values of a Dataset from a paired ImagePlus. Assumes the Dataset and ImagePlus have compatible dimensions and that the data planes are not directly mapped. Gets values via ImageProcessor::getf(). In cases where there is a narrowing of data into modern ImageJ types the data is range clamped. Does not change the Dataset's metadata.
        Specified by:
        updateDataset in interface DataHarmonizer

      • updateLegacyImage

        public void updateLegacyImage(Dataset ds,
                              ImagePlus imp)
        Assigns the data values of an ImagePlus from a paired Dataset. Assumes the Dataset and ImagePlus are not directly mapped. It is possible that multiple modern ImageJ axes are encoded as a single set of channels in the ImagePlus. Sets values via ImageProcessor ::setf(). Some special case code is in place to assure that BitType images go to legacy ImageJ as 0/255 value images. Does not change the ImagePlus' metadata.

        updateLegacyImage in interface DataHarmonizer
