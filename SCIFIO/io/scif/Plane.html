    public interface Plane
extends Contextual
    Top-level interface for all Plane representations in SCIFIO.

    Planes are X,Y slices of pixel data from a parent image, and potentially include a ColorTable if that parent uses indexed color.

    This interface guarantees that a Plane will have a ColorTable, and the pixel data of a Plane can ultimately be converted to a byte[].
