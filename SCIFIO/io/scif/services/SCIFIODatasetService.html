    Comparable<Prioritized>, DatasetService, ImageJService, Contextual, Disposable, Identifiable, Initializable, Locatable, Logged, HasPluginInfo, RichPlugin, SciJavaPlugin, Prioritized, Service, Versioned
    Deprecated. 
    Use DatasetIOService for the canOpen(String), canSave(String), save(net.imagej.Dataset, java.lang.String), open(java.lang.String) and revert(Dataset) methods. Use DatasetService for all other methods.

    @Deprecated
public class SCIFIODatasetService
extends AbstractService
implements DatasetService
    This is a temporary service designed to supercede DefaultDatasetService. Specifically it implements the deprecated API from DatasetService by delegating to the DatasetIOService while the remaining methods are deferred to the next-highest priority DatasetService.
    Author:
    Mark Hiner
