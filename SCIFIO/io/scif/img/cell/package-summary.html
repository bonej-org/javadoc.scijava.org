Package io.scif.img.cell Description

Provides a Reader backed CellImg implementation and supporting classes.

These classes dynamically load tiles (Cells) of images as requested, allowing exceedingly large images to be read immediately.

The SCIFIOCellCache is the entry point for loading Cells. Cell positions are requested from the cache, which are then loaded if they do not exist already. The cache can be memory or disk-based.
